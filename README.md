# Создать приложение "Дневник путешествий"

1. Необходимо создать web-сайт, используя любой стек технологий Web-разработки
2. Создать пользователя системы 
3. Создать для пользователя функцию записи своего путешествия
4. Создать функцию просмотра путешествий других пользователей
5. Добавить перечисленные функциb о путешествии:
    * Местоположение (с привязной к геопозиции)
    * Изображение мест
    * Стоимость путешествия
    * Места культурного наследия
    * Места для посещения
    * Оценка удобства передвижения / безопасности / населенности / растительности  
6. Опишите подробный анализ по выполненной задаче (не менее 7 пунктов)
7. Опишите рекомендации по устранению выявленных ошибок в ходе выполнения задачи

# Cтек
 * Frontend: React.js 19.1.0
 * Styling: CSS3
 * Backend: JSON Server (для имитации REST API)
 * HTTP Client: Axios

# Установка и запуск
 * Клонировать репозиторий

    git clone https://github.com/edraider8/case-5

    cd case-5

 * Установка зависимостей:

   npm install


 * Запуск приложения:

   npm run dev

   Эта команда запустит одновременно:
   - JSON Server на порту 3001 (API)
   - React приложение на порту 3000
